# Encrypted Timeseries App

A real-time data streaming system that handles encrypted messages securely.

## Quick Start
Run the entire application with a single command:
```bash
./start.sh
```
This will:
1. Build and start all services (Mongo, Listener, Emitter, Frontend).
2. Automatically open the dashboard in your browser.

- **Dashboard**: http://localhost:8080
- **Listener API**: http://localhost:3001
